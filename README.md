# Tajir POS (Optimized)

This repository contains the optimized and modularized version of the Tailor POS system.

## Documentation
See the `docs/` directory for detailed documentation on:
- Modularization Strategy
- API Structure
- Deployment Guides
- Feature Plans

## Key Changes
- **Modular Architecture**: Backend split into Blueprints (`api/`), Frontend into Modules (`static/js/modules/`).
- **Performance**: Optimized database queries and asset loading.
- **Structure**: Clean separation of concerns (Templates, API, Logic).
